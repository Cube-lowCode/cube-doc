## 快速上手

* 设置全局环境

```bash
npm config set registry https://repo.yupaopao.com/artifactory/api/npm/npm-repo/
npm install -g @isaac/cli
```

* 安装依赖

```bash
npm i
```

* 启动本地开发服务

```bash
npm run dev
```
